# 许皓轩的个人空间

欢迎来到我的个人空间项目！这是一个展示我学习成果、分享作品和兴趣爱好的个人网站。

## 项目结构

- `index.html`：网页的主文件，包含个人信息、导航栏、主要内容区域和页脚。
- `css/style.css`：用于网页样式的CSS文件。
- `js/main.js`：用于网页交互效果的JavaScript文件。
- `images/`：存放网站使用的图片资源。

## 使用说明

1. **克隆或下载项目**：将项目克隆到本地或下载ZIP文件并解压。
2. **打开网页**：在浏览器中打开 `index.html` 文件。
3. **浏览内容**：使用导航栏访问不同的页面部分，了解我的学习经历、技术作品、创意设计和生活兴趣。

## 技术栈

- HTML5
- CSS3
- JavaScript
